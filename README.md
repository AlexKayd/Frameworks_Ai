# Machine Learning Research Project

Лабораторные работы по исследованию алгоритмов машинного обучения. Мной было проведено исследование шести основных алгоритмов машинного обучения на двух реальных практических задачах:

- **Классификация**: Сегментация клиентов (мультиклассовая)
- **Регрессия**: Предсказание возраста крабов
 
## Наборы данных

### Классификация: Customer Segmentation
- **Источник**: [Kaggle - Customer Segmentation](https://www.kaggle.com/datasets/vetrirah/customer)
- **Целевая переменная**: `Segmentation` (4 класса: A, B, C, D)
- **Задача**: Мультиклассовая классификация клиентов для персонализации маркетинга
- **Практическая ценность**: Улучшение таргетирования рекламы и предложений

### Регрессия: Crab Age Prediction  
- **Источник**: [Kaggle - Crab Age Prediction](https://www.kaggle.com/datasets/sidhus/crab-age-prediction)
- **Целевая переменная**: `Age` (возраст краба в месяцах)
- **Задача**: Регрессия для оптимизации времени сбора урожая
- **Практическая ценность**: Максимизация прибыли крабовых ферм

## Метрики качества

### Классификация
- **Accuracy** - общая точность
- **Precision, Recall, F1-Score** - по каждому классу

### Регрессия
- **MAE** (Mean Absolute Error) - средняя абсолютная ошибка
- **RMSE** (Root Mean Square Error) - корень из среднеквадратичной ошибки  
- **R2** (R-squared) - коэффициент детерминации
